## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/MercifulB/quantobooks-reimagined-glow.git
```

2. Install dependencies:
```bash
cd quantobooks-reimagined-glow
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
